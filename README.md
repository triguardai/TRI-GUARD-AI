# TRI-GUARD-AI

Landing page prototype untuk TriGuard AI, solusi AI escrow untuk membantu mencegah penipuan transaksi online dengan proteksi dana, verifikasi rekening, blacklist scammer, dan simulasi pelacakan resi.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Menjalankan Project

```bash
npm install
npm run dev
```

## Build Production

```bash
npm run build
```
